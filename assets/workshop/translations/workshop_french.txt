[h1]Présentation :[/h1]
Outils destinés à l'hôte en mode multijoueur pour la gestion des parties multijoueurs ; en mode multijoueur, ils ne peuvent être activés et utilisés que par l'hôte. Ils incluent des journaux de toutes les actions effectuées, visibles par l'ensemble du salon via le [b]Journal des actions du mod[/b]. Ils peuvent également être utilisés en mode solo comme menu de triche.

Ce mod désactive également temporairement Hellenism, connu pour causer des problèmes de désynchronisation en multijoueur. Vous pouvez consulter l'état d'avancement du bug [url=https://forum.paradoxplaza.com/forum/threads/constant-multiplayer-desync-from-platonic-revival-movement-plus-workaround.1923989/]ici.[/url]

[h1]Mode d'emploi :[/h1]
[olist]
 [*] Ouvrez le menu du mod, soit en appuyant sur la touche « m », soit en jeu en appuyant sur Échap -> menu du mod
 [*] Dans l’onglet « Community Mod Framework », activez « Enable Host-Only Tools » (en mode solo, cette option s’intitule « Enable Unrestricted Tools »). Notez que cette fonctionnalité n’est disponible que pour l’hôte en mode multijoueur.
 [*] Les outils seront alors disponibles sous forme d’actions diplomatiques pour l’hôte ; il suffit d’ouvrir le menu « Diplomatie » pour les trouver dans la catégorie « Outils de l’hôte ».
 [*] Vous pouvez configurer l’outil de l’hôte depuis l’onglet « Outil de l’hôte » du menu du mod.
[/olist]
- Tant que cette option est active, l’hôte disposera d’une portée diplomatique illimitée et pourra donc utiliser l’outil partout ; il est donc recommandé de la désactiver après utilisation.

[h1]Comment consulter le journal :[/h1]
[olist]
 [*] Ouvrez le menu des mods, soit en appuyant sur la touche « m », soit en jeu en appuyant sur Échap -> menu des mods
 [*] Rendez-vous dans l'onglet « Community Mod Framework ».
 [*] Cliquez sur le bouton « Ouvrir » situé à côté de « Journal des actions du mod »
[/olist]
- Notez qu'en mode multijoueur, le journal ne peut pas être effacé, même par l'hôte.

[h1]Compatibilité :[/h1]
[list]
 [*]Non compatible avec le mode Ironman ni les succès.
 [*]Entièrement compatible avec les sauvegardes.
 [*]Devrait fonctionner avec n'importe quel mod.
 [*]Nous pouvons ajouter des correctifs de compatibilité sur demande.
[/list]
